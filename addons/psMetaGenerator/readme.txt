-- Installation Instrucitons --

To install, upload the addon folder "psMetaGenerator" to the "addons" folder of your Geodesic installation. For example: http://www.mysite.com/addons/psMetaGenerator

Log into the administrator panel for your Geodesic software (e.g. http://www.mysite.com/admin ) and click 'ADDONS', then 'Manage Addons' in the left navigation panel.

Under 'Not Installed Addons' you will see 'PS Meta Generator' with an 'Install' link to the left. Click that link and once installation has finished, refresh the page.

'PS Meta Generator' will now be listed under 'Disabled Addons'. Click 'Enable' to start using your Google Sitemap generator. 




-- Usage Instructions --

In the left navigation panel under 'ADDONS', there is now an 'PS Meta Generator' option with a link to 'Configure'. Click this to enter your meta-tag configuration. This is the default title, keywords and description to use throughout the site if none are specified for the given page or category.

Specify specific titles, keywords and descriptions for your categories and pages.

Make sure any current <title></title> and <meta name="description" /> and <meta name="keywords" /> tags are removed from your templates as these will now be added for you.
