Export a Language
To export a language, follow these instructions:
  1. Save your language by clicking the 'Download' below.
  2. Open the resulting text.csv file in a spreadsheet editor such as Microsoft Excel or OpenOffice Calc.
  3. Translate each piece of text that is located in the 'Text' column.
  4. Save as text.csv. The file must be saved using the Comma-Separated-Values (CSV) format.
  5. Upload (FTP) the translated text.csv file to the 'admin' directory of your software on the server.
  6. Locate the language in the Languages Menu that you would like to import text for. You might have to create a new language first.
  7. Click the 'Import' link next to the language you would like to import your translated text into.
